📦 Installation Guide

📱 Termux (Android)

```bash
pkg update -y && pkg upgrade -y
pkg install python -y
pkg install python-pip -y
pkg install git -y
git clone https://github.com/shahid2005a/DGTL-DELETE-.git
cd DGTL-DELETE-
python main.py
```

⚡ Single Command Installation

```bash
yes | pkg update && yes | pkg upgrade && yes | pkg install python python-pip git -y && echo "y" | termux-setup-storage && git clone https://github.com/shahid2005a/DGTL-DELETE-.git && cd DGTL-DELETE- && python main.py
```

🐧 Linux (Ubuntu/Debian/Kali/Parrot)

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip git -y
git clone https://github.com/shahid2005a/DGTL-DELETE-.git
cd DGTL-DELETE-
sudo python3 main.py
```

🟡 Linux (Fedora/RHEL)

```bash
sudo dnf update -y
sudo dnf install python3 python3-pip git -y
git clone https://github.com/shahid2005a/DGTL-DELETE-.git
cd DGTL-DELETE-
sudo python3 main.py
```

🔵 Windows (CMD as Administrator)

```cmd
python -m pip install --upgrade pip
git clone https://github.com/shahid2005a/DGTL-DELETE-.git
cd DGTL-DELETE-
python main.py
```

🟣 MacOS

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python3 git
```

⚪ Alpine Linux
